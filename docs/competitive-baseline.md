# Competitive Baseline

ProofOps is not a general agent orchestration framework. It does not compete directly with LangGraph, CrewAI, AutoGen, OpenHands, or Codex.

ProofOps focuses on release governance:

- maturity target presets
- architecture readiness
- real-boundary evidence
- release coverage matrix
- productized repair loop
- final report
- GO / CONDITIONAL-GO / NO-GO / BLOCKED decisions

Use ProofOps when an agent product needs evidence-backed movement from demo, alpha, beta, or RC toward GA.
